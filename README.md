### Prerequisites

Ensure you have the following installed on your machine:

- **[Node.js](https://nodejs.org/)**: v14.x or higher
- **[npm](https://www.npmjs.com/)**: v6.x or higher (or `yarn` if you prefer)

- **[class-variance-authority](https://www.npmjs.com/package/class-variance-authority)**: v0.7.0 ‚Äì Manages variant classes for styling

- **[swiper](https://www.npmjs.com/package/swiper)**: v11.1.14 ‚Äì Swiper library for touch sliders
